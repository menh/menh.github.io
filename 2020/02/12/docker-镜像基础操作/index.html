docker-镜像基础操作

Posted on 2020-02-12 Edited on 2020-02-20 In Docker

  1. 列出镜像
    docker images

  2. 查看镜像
    docker inspect

  3. 删除镜像
    docker rmi

  4. 查找镜像
    docker search

  5. 拉去镜像
    docker pull

  6. 推送镜像
    docker push

例子docker push qlassroom_ai_dev

  1. 构建镜像
    docker commit

利用dockerfile 构建镜像
docker build

  1. dockerfile 构建镜像

  2. docker tag 规范镜像名字
    docker tag qlassroom_ai_dev:latest 872204778/qlassroom_dev:v1
    qlassroom_ai_dev 为原镜像名称
    latest: 为原镜像标签

872204778： 用户名
qlassroom_dev：新镜像名
v1:新标签
